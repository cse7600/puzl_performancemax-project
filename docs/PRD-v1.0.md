## 1. 프로젝트 개요

### 비전
> 주니어 퍼포먼스 마케터도 시니어처럼 의사결정할 수 있는 생산성 도구

퍼포먼스 마케터가 하루에 소비하는 시간 중 **60~70%는 반복적인 데이터 수집·분석·판단**에 쓰인다.
이 플랫폼은 그 과정을 자동화하여 마케터가 "창의적인 전략 수립"에 집중할 수 있게 한다.

### 타겟 사용자
| 사용자 유형 | 특징 | 주요 니즈 |
|------------|------|-----------|
| 주니어 마케터 (1~3년차) | 판단 기준 불명확, 데이터 해석 어려움 | 명확한 액션 가이드 |
| 시니어 마케터 (4년↑) | 다수 계정 동시 관리, 시간 부족 | 자동화·일괄 처리 |
| 마케팅 에이전시 | 복수 클라이언트 계정 운영 | 멀티 계정 통합 뷰 |

### 지원 플랫폼 (Phase별)
- **Phase 1 (진행 중):** Meta, 네이버 검색광고, Google Ads — API 키 모두 확보됨
- **Phase 2:** 멀티플랫폼 통합 뷰, API 자동 수집, Kakao Moment/Keyword
- **Phase 3:** AI 추천, 자동 입찰 조정 연동

### 보유 API 키 현황
| 플랫폼 | 상태 | 액세스 수준 | 비고 |
|--------|------|-----------|------|
| Meta Graph API | ✅ 확보 | User Token (60일 만료) | System User Token 전환 권장 |
| 네이버 검색광고 API | ✅ 확보 | 정식 키 | Customer ID 2441318 |
| Google Ads API | ✅ 확보 | 기본 액세스 (Basic) | 내부 툴 운영 충분, Standard 신청 가능 |
| Supabase | ✅ 확보 | Anon Key | Service Role Key 미설정 — 서버 쓰기 시 필요 |
| Kakao 광고 API | ❌ 미확보 | - | 공식 파트너 여부 확인 필요 |
