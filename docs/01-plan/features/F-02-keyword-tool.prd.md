## 1. 기능 개요

### 문제 정의
퍼포먼스 마케터가 키워드 광고를 집행할 때 반드시 알아야 하는 정보:
1. **이 키워드에 사람이 얼마나 검색하는가?** (검색량)
2. **경쟁이 치열한가?** (경쟁도)
3. **경쟁자들이 이 키워드에서 어떻게 광고하는가?** (광고주 순위 / 소재 내용)
4. **적정 입찰가는 얼마인가?** (CPC 추정)

현재 이 정보를 얻으려면:
- 네이버 검색광고 시스템 → 키워드 플래너 (5개씩만 조회, 로그인 필요)
- Google Keyword Planner (별도 접속, 단위가 다름)
- 네이버 검색 결과 직접 확인 (수작업)

**하나의 화면에서 전부 해결할 수 있어야 한다.**

## 2. 현재 구현 기능 (v1.0)

### 2.1 키워드 검색량 조회
- **입력:** 키워드 복수 입력 (쉼표 또는 줄바꿈 구분, 최대 100개)
- **출력:**

| 항목 | 출처 | 설명 |
|------|------|------|
| 네이버 PC 검색수 | 네이버 검색광고 API | 월간 PC 환경 검색 건수 |
| 네이버 모바일 검색수 | 네이버 검색광고 API | 월간 모바일 검색 건수 |
| 네이버 총합 | 자체 계산 | PC + 모바일 합산 |
| 구글 검색량 | Google Ads API (Keyword Ideas) | 월간 구글 검색 건수 |
| 네이버 경쟁도 | 네이버 검색광고 API | 낮음/중간/높음 |
| 구글 경쟁도 | Google Ads API | 낮음/중간/높음 |
| 월간 노출광고수 (avgDepth) | 네이버 검색광고 API | 해당 키워드에 광고 노출된 평균 광고주 수 |

- **정렬:** 총 검색수 내림차순 기본 정렬
- **내보내기:** CSV 다운로드

### 2.2 네이버 광고주 순위 모니터
- 등록 키워드별 광고주 순위를 주기적으로 수집
- PC / 모바일 탭 전환
- 직전 수집 대비 변동 표시: NEW / 제거 / ▲ / ▼ / ━
- 광고 카드 뷰: 제목, 설명, 서브링크 수, 이미지 수 확인

## 3. 데이터 소스 및 API

### 3.1 네이버 검색광고 API
- **엔드포인트:** `https://api.naver.com/keywordstool`
- **인증:** HMAC-SHA256 서명 (timestamp + method + URI)
- **환경변수:** `NAVER_AD_API_KEY`, `NAVER_AD_CUSTOMER_ID`, `NAVER_AD_SECRET_KEY`
- **제약:** 1회 요청당 최대 5개 키워드 → 배치 처리 (300ms 딜레이)

### 3.2 Google Ads API
- **방식:** REST API (`generateKeywordIdeas`)
- **인증:** OAuth2 토큰
- **환경변수:** `GOOGLE_ADS_DEVELOPER_TOKEN`, `GOOGLE_ADS_CUSTOMER_ID`, `GOOGLE_ADS_ACCESS_TOKEN`

### 3.3 네이버 SERP 스크래핑
- GitHub Actions + Playwright 기반
- 키워드당 PC/Mobile 검색 결과 스크린샷 + 광고주 목록 추출
- Supabase에 스냅샷 저장

## 5. 사용성 설계

### 5.1 현재 UX
1. 키워드 입력창에 여러 키워드를 줄바꿈으로 입력
2. "검색량 조회" 버튼 클릭
3. 표 형태로 결과 표시 (총 검색수 내림차순 정렬)
4. CSV 다운로드 가능

### 5.2 개선 필요 사항 (다음 버전)
- **키워드 그룹 저장:** 자주 조회하는 키워드 세트를 저장하고 재사용
- **트렌드 비교:** 이번 달 vs 지난 달 검색량 변화 표시
- **관련 키워드 확장:** 입력 키워드의 연관/확장 키워드 자동 추천
- **경쟁자 분석:** 특정 키워드에 노출되는 경쟁자 광고 소재 비교

## 6. 확장 방향 (Keyword Intelligence v2.0)

### 6.1 키워드 기회 스코어링

각 키워드를 다음 기준으로 점수화:

| 항목 | 가중치 | 높은 점수 기준 |
|------|--------|--------------|
| 검색량 | 40% | 월 1만 이상 |
| 경쟁도 | 30% | 낮음 (기회) |
| 검색량 증가 추세 | 20% | 전월 대비 +10% 이상 |
| 광고주 밀도 (avgDepth) | 10% | 적을수록 기회 |

→ 최종 **기회 점수 (0~100)** 표시

### 6.2 입찰가 추천
- 네이버: 최소/권장/공격적 CPC 3구간 제시
- 근거: 검색량 × 경쟁도 × 광고주 수 복합 계산

### 6.3 멀티플랫폼 키워드 비교
| 키워드 | 네이버 검색량 | 구글 검색량 | N 경쟁도 | G 경쟁도 | 추천 플랫폼 |
|--------|-------------|-----------|---------|---------|------------|
| 퍼포먼스마케팅 | 12,400 | 3,200 | 중간 | 낮음 | 네이버 |

### 6.4 캠페인 키워드 관리
- 모니터링 키워드를 캠페인/광고그룹 단위로 태깅
- 태그별 검색량 합산, 경쟁도 분포 요약
- 키워드 추가/제거 제안 (성과 연계)
